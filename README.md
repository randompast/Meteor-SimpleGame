Meteor-SimpleGame
=================

###Installation - Running

1. git clone https://github.com/randompast/Meteor-SimpleGame.git
2. cd Meteor-SimpleGame
3. meteor
4. Browser: localhost:3000
5. Create account (left)
6. Arrow keys to move
